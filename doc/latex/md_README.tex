Projekt obsahuje předpřipravený grafický instalátor.
\begin{DoxyEnumerate}
\item Přejděte do složky {\ttfamily install/}.
\item Spusťte soubor {\ttfamily IVS\+\_\+calculator\+\_\+setup.\+exe} a postupujte podle instrukcí průvodce.
\item Aplikaci lze poté spustit přímo z plochy nebo z nabídky Start.
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md5}{}\doxysubsection{\texorpdfstring{Pro uživatele Linuxu (a vývojáře)}{Pro uživatele Linuxu (a vývojáře)}}\label{md_README_autotoc_md5}
Projekt využívá nástroj {\ttfamily make} pro automatizaci běžných úloh. V kořenovém adresáři repozitáře lze spustit následující příkazy\+:

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make install}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Nainstaluje všechny potřebné závislosti (především framework Flet) nezbytné pro běh aplikace.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make build}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Vytvoří spustitelnou verzi kalkulačky pro Linux.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make run}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Spustí samotnou aplikaci kalkulačky s grafickým rozhraním.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make test}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Spustí sadu automatizovaných testů pro ověření správné funkčnosti matematické knihovny.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make doc}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Vygeneruje vývojářskou dokumentaci zdrojového kódu (Doxygen) do složky {\ttfamily doc/}.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make pack}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Zabalí celý repozitář do {\ttfamily .zip} archivu ve formátu požadovaném pro finální odevzdání projektu.

\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily make clean}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} Odstraní všechny dočasné soubory, meziprodukty překladu, vygenerovanou dokumentaci a instalační archivy.\hypertarget{md_README_autotoc_md6}{}\doxysection{\texorpdfstring{Struktura repozitáře}{Struktura repozitáře}}\label{md_README_autotoc_md6}

\begin{DoxyItemize}
\item {\ttfamily src/} -\/ Zdrojové kódy aplikace, matematické knihovny a grafického rozhraní. Obsahuje i Skripty s automatizovanými testy matematické knihovny.
\item {\ttfamily doc/} -\/ Uživatelský manuál a technická dokumentace.
\item {\ttfamily install/} -\/ instalační soubor pro operační systém windows
\item {\ttfamily Makefile} -\/ Skript definující cíle pro sestavení, spuštění a testování projektu. 
\end{DoxyItemize}
